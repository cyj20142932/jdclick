# JDHelper 界面风格重新设计规格文档

**日期：** 2026-04-10
**主题：** 界面风格设计及交互重构
**设计模式：** 仪表盘式首页

---

## 1. 设计目标

- **视觉风格：** 实用主义 + 简约现代，蓝绿渐变配色
- **交互原则：** 效率优先，减少操作步骤
- **核心场景：** 京东定时抢购 + 通用自动化

---

## 2. 配色方案

### 2.1 色彩定义

| 用途 | 颜色 | Hex |
|------|------|-----|
| 主色-渐变起始 | 蓝绿色 | #00B4DB |
| 主色-渐变结束 | 深蓝色 | #0083B0 |
| 背景-深色 | 深蓝黑 | #1A1A2E |
| 背景-浅色 | 浅灰白 | #F5F5F7 |

| 卡片-浅色 | 纯白色 | #FFFFFF |

| 状态-警告 | 橙色 | #FF9800 |
| 状态-异常 | 红色 | #F44336 |
| 文字-主色 | 深灰 | #212121 |
| 文字-次色 | 中灰 | #757575 |

### 2.2 主题适配

- 默认使用深色主题，符合科技感定位

- 渐变色仅用于顶部状态栏和关键按钮

## 4. 首页详细设计

### 4.1 顶部状态栏

- **高度：** 56dp
- **背景：** 蓝绿渐变 (`#00B4DB` → `#0083B0`)
- **内容：**
  - 左侧：NTP时间 (HH:mm:ss)
  - 中间：毫秒显示 (xxx)
  - 右侧：下次定时点击倒计时 (⏱ MM:SS)
- **字体：** 20sp 等宽字体，时间数字突出显示
- **特点：** 始终固定在顶部，切换Tab时保持可见

### 4.2 状态卡片区

- **布局：** 2x2 网格排列
- **卡片尺寸：** 自适应，间距 12dp
- **卡片内容：**
  - 图标 (32dp)
  - 标题
  - 状态文字
  - 状态指示点/开关
- **交互：**
  - 点击卡片 → 跳转对应设置页面
  - 开关类 → 直接Toggle

### 4.3 快捷操作区

- **布局：** 横向排列的按钮组
- **按钮样式：** 圆角按钮，带图标+文字
- **操作：**
  - 时间同步：立即同步NTP
  - 点击延迟：弹出设置对话框
  - 新建任务：跳转到任务创建页（如有）

## 5. 设置页优化

### 5.1 重排序逻辑

按使用频率排序：
1. 权限管理（最高频）
2. 点击延迟设置
3. 时间同步设置
4. 其他设置

### 5.2 分组精简

| 分组 | 包含项 |
|------|--------|
| 权限管理 | 无障碍服务、悬浮窗权限、通知权限 |
| 点击设置 | 点击延迟、点击持续时间、毫秒格式 |
| 历史记录 | 记录开关、历史查看、清除数据 |
| 关于 | 版本信息、使用帮助 |

### 5.3 交互优化

- 开关类设置：滑块 + 即时保存，无确认按钮
- 数值类设置：点击弹出数字键盘输入
- 分组标题：可折叠，减少视觉负担

### 6.2 空状态

## 7. 交互细节

### 7.1 动画效果

- 卡片点击：轻微缩放反馈 (scale 0.98)
- 开关切换：平滑过渡动画 (300ms)
- 页面切换：淡入淡出 (200ms)

### 7.2 状态反馈

- 操作成功：Toast 提示 + 卡片高亮闪烁
- 操作失败：Toast 提示 + 卡片红色边框
- 加载中：卡片内显示 CircularProgressIndicator

### 7.3 防误触

- 删除操作：保留确认对话框
- 危险操作（如清除数据）：二次确认

## 8. 技术实现要点

### 8.1 Theme 扩展

```kotlin
// 新增颜色
val BlueGreenStart = Color(0xFF00B4DB)
val BlueGreenEnd = Color(0xFF0083B0)
val DarkBackground = Color(0xFF1A1A2E)
val DarkCard = Color(0xFF252540)

// 修改默认主题
private val DarkColorScheme = darkColorScheme(
    primary = BlueGreenStart,
    secondary = BlueGreenEnd,
    background = DarkBackground,
    surface = DarkCard
)
```

### 8.2 顶部状态栏实现

- 使用 `Scaffold` 的 `topBar` 插槽
- 固定位置：`TopAppBar` + `position = TopAppBarDefaults.pinnedScrollBehavior()`
- 时间更新：ViewModel 中 Flow 每秒更新

### 8.3 卡片组件复用

- 创建 `StatusCard` 通用组件
- 参数：icon, title, status, isEnabled, onToggle, onClick
